正则表达式

正则表达式的功能十分强大，绝大多数的字符串都能通过它来进行表示，而且绝大多数语言都支持值得我们去学习。在这里我用Python语言来演示。

普通字符串的匹配

字母、数字、汉子、下划线、以及没有特殊定义的符号，都是“普通字符”。正则表达式中的普通字符，在匹配时，只匹配与自身相同的一个字符。　

运行结果

必须和表达式完全一样才能被匹配

元字符

正则表达式中使用了很多元字符，用来表示一些特殊的含义或功能。

表达式 匹配
. 小数点可以匹配除了换行符\n以外的任意一个字符
｜ 逻辑或操作符
[] 匹配字符集中的一个字符
[^] 字符集求反，也就是上面的反操作。尖号必须在括号的最前面
\ 对紧跟其后的一个字符进行转义
() 对表达式进行分组，将圆括号内的内容当做一个整体，并获得匹配的值

预定义匹配字符集

正则表达式中的一些表示方法，可以同时匹配某个预定义字符集中的任意一个字符。

\d 任意一个数字，０～９中的任意一个

\W \w的反集

\S \s的反集

Java中printf小记

在Java中，print就是一般的标准输出，但是每个print语句之间不会换行(可用转义符\n做出换行效果)这时的效果就和单独使用println一样。
对于像我这样的Java初学者printf就略显陌生了，下面就是对printf的简单介绍。

printf

printf的作用是进行格式输出，如果你学习过C语言，相信你应该会很熟悉
在使用Java时常常会出现这种情况

若果还有其他的变量要插入到字符串中，就会添加更多+号，使编写很繁琐

使用printf就可以很好的避免这种问题

String name = LiSi;
int id = 002;
String sex = 女;

System.out.printf("name is：%s id is：%s sex is：%s",name,id.sex);

参照Java API的定义如下：
‘d’ 整数结果被格式化为十进制整数
‘o’ 整数结果被格式化为八进制整数
‘x’, ‘X’ 整数结果被格式化为十六进制整数
‘e’, ‘E’ 浮点结果被格式化为用计算机科学记数法表示的十进制数
‘f’ 浮点结果被格式化为十进制数
‘g’, ‘G’ 浮点根据精度和舍入运算后的值，使用计算机科学记数形式或十进制格式对结果进行格式化
‘a’, ‘A’ 浮点结果被格式化为带有效位数和指数的十六进制浮点数
‘n’ 换行
