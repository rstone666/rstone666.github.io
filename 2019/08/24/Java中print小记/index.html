Java中printf小记

在Java中，print就是一般的标准输出，但是每个print语句之间不会换行(可用转义符\n做出换行效果)这时的效果就和单独使用println一样。
对于像我这样的Java初学者printf就略显陌生了，下面就是对printf的简单介绍。

printf

printf的作用是进行格式输出，如果你学习过C语言，相信你应该会很熟悉
在使用Java时常常会出现这种情况

若果还有其他的变量要插入到字符串中，就会添加更多+号，使编写很繁琐

使用printf就可以很好的避免这种问题

String name = LiSi;
int id = 002;
String sex = 女;

System.out.printf("name is：%s id is：%s sex is：%s",name,id.sex);

参照Java API的定义如下：
‘d’ 整数结果被格式化为十进制整数
‘o’ 整数结果被格式化为八进制整数
‘x’, ‘X’ 整数结果被格式化为十六进制整数
‘e’, ‘E’ 浮点结果被格式化为用计算机科学记数法表示的十进制数
‘f’ 浮点结果被格式化为十进制数
‘g’, ‘G’ 浮点根据精度和舍入运算后的值，使用计算机科学记数形式或十进制格式对结果进行格式化
‘a’, ‘A’ 浮点结果被格式化为带有效位数和指数的十六进制浮点数
‘n’ 换行
